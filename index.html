    Salve

    This is a website that is used to talk back and forth between users randomly assigned to each other similar to omegle.     Connect Four

    This game was created to learn how to develop server dependent websites and, in this case, specifically GET and POST requests to a server. The way this game works is it utilizes a lobby system to create connections between two players directly. Players play back and forth and are not allowed to place more than one tile at a time.

    Unity

    These exercises are to showcase different interactive programs I worked on in Unity through a semester. They range in simplicity and interactivity.

    • Testing out terrain and prefab creation
    • Using prefabs, terrain and particle effects
    • Created a clock whose hand can follow the mouse
    • Move around using arrow keys and see if you collide with the rocks
    • Watch blocks seek different things and flee different objects

    D&D Spellbook Creator

    This project uses the D&D API, localstorage, and Firebase to create custom Spellbooks for different characters that you can create. The Firebase side of the project allows for different people to upload and download different spellbooks. This has been very useful to me as I love to play D&D but writing all the spells one character has is horrible. To be able to sort through a character's spells and look them up by name is nice and convenient.
    Make some Spellbooks!
